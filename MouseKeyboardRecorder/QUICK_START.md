## 安装与运行

### 方式一：直接运行（需要 .NET 8.0）

```bash
# 1. 克隆或下载项目
cd MouseKeyboardRecorder

# 2. 运行程序
dotnet run
```

### 方式二：发布为独立应用

```bash
# 发布为单文件可执行程序
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true

# 运行发布的程序
.\bin\Release\net8.0-windows\win-x64\publish\MouseKeyboardRecorder.exe
```

### ⚠️ 重要提示

**必须以管理员身份运行！** 否则无法捕获全局输入。

## 基本录制

### 步骤 1：开始录制

点击工具栏上的 **🔴 录制** 按钮，或按下 **F9** 键。

状态栏会显示 "🔴 正在录制..."，表示录制已开始。

### 步骤 2：执行操作

执行您想要录制的鼠标和键盘操作：

- **鼠标操作**：
  - 移动鼠标到指定位置
  - 左键/右键/中键点击
  - 滚轮滚动

- **键盘操作**：
  - 按下任意按键
  - 输入文本
  - 使用快捷键组合

### 步骤 3：停止录制

点击 **⏹️ 停止** 按钮，或再次按下 **F9** 键。

录制的操作会显示在下方的列表中。

## 播放操作

### 基本播放

1. 确保已有录制的操作
2. 点击 **▶️ 播放** 按钮，或按下 **F10** 键
3. 程序会自动重现所有录制的操作

### 调整播放速度

- 拖动 **播放速度** 滑块
- 范围：0.1x（慢速）到 5.0x（快速）
- 默认：1.0x（正常速度）

### 循环播放

- 勾选 **启用循环** 复选框
- 在 **循环次数** 中输入次数
- 输入 0 表示无限循环

### 暂停与恢复

- 播放中点击 **⏸️ 暂停** 或按 **F10** 暂停
- 再次点击或按 **F10** 恢复播放
- 点击 **⏹️ 停止** 终止播放

### 人类行为模拟

建议保持 **模拟人类操作** 选项勾选：
- 添加随机延迟（±5-10%）
- 使操作更自然
- 避免被检测为机器人

## 编辑录制

### 查看操作详情

操作列表显示以下信息：
- **序号**：操作在序列中的位置
- **操作类型**：鼠标/键盘/滚轮等
- **描述**：操作的详细说明
- **坐标/键码**：位置或按键信息
- **延迟**：与上一个操作的时间间隔
- **时间戳**：操作发生的时间

### 修改操作

1. 在列表中点击选择要编辑的操作
2. 点击 **✏️ 编辑** 按钮
3. 在弹出的对话框中修改参数：
   - 操作类型
   - 坐标位置（鼠标操作）
   - 键码（键盘操作）
   - 延迟时间
4. 点击 **确定** 保存修改

### 删除操作

1. 选择要删除的操作
2. 点击 **❌ 删除** 按钮
3. 确认删除

### 清空所有

点击 **🗑️ 清空** 按钮删除所有操作。

## 保存与加载

### 保存录制

1. 点击 **💾 保存** 按钮
2. 选择保存位置
3. 输入文件名（默认：录制_YYYYMMDD_HHMMSS.json）
4. 点击保存

录制文件使用 JSON 格式，可以：
- 在不同电脑间分享
- 使用文本编辑器查看
- 通过代码解析处理

### 加载录制

1. 点击 **📂 加载** 按钮
2. 选择之前保存的 .json 文件
3. 点击打开

加载后会显示：
- 操作数量
- 总时长
- 录制时的屏幕分辨率

## 💡 使用技巧

### 高效录制

1. **提前规划**：录制前先想清楚操作步骤
2. **稳定执行**：操作时保持稳定的速度
3. **适当停顿**：关键步骤后添加短暂停顿
4. **测试回放**：录制后立即测试播放效果

### 循环任务

适合使用循环播放的场景：
- 批量文件处理
- 重复性数据录入
- 定时刷新操作
- 游戏挂机脚本

### 注意事项

1. **屏幕分辨率**：录制和播放时保持相同分辨率
2. **窗口位置**：确保目标窗口在相同位置
3. **权限问题**：始终以管理员身份运行
4. **系统安全键**：无法录制 Ctrl+Alt+Del 等系统键

## ❓ 常见问题

**Q: 为什么无法开始录制？**  
A: 请确保以管理员身份运行程序。

**Q: 播放时位置不准确？**  
A: 确保录制和播放时的屏幕分辨率一致，且目标窗口位置相同。

**Q: 可以录制多长时间？**  
A: 理论上无限制，但建议单次录制不超过 30 分钟，以免文件过大。

**Q: 文件保存在哪里？**  
A: 默认保存到文档目录，可在保存对话框中选择其他位置。
