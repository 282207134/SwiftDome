# 01 · 基础知识

参考：[Swift 中文文档 · 基础部分](https://doc.swiftgg.team/documentation/the-swift-programming-language/thebasics)

## 学习目标

- 理解常量 (`let`) 与变量 (`var`) 的区别
- 熟悉 Swift 的类型系统、可选类型与类型推断
- 掌握字符串、数字、布尔、元组等基础数据结构
- 会使用断言、先决条件以及打印调试信息

## 核心概念速查

| 概念 | 说明 | 示例 |
| --- | --- | --- |
| 常量 / 变量 | 不可变 / 可变的存储单元 | `let name = "Swift"`，`var count = 0` |
| 类型推断 | 编译器根据字面量自动判定类型 | `let pi = 3.14` → `Double` |
| 可选类型 | 用 `?` 表示“有值或为 nil” | `var score: Int? = nil` |
| 可选绑定 | 使用 `if let` / `guard let` 安全解包 | `if let number = Int(text)` |
| 元组 | 将多个值组合为一个复合值 | `let http404 = (404, "Not Found")` |
| 字符串插值 | 将变量嵌入字符串 | `"结果：\(value)"` |
| 区间 | `1...5`（闭区间）、`..<`（半开） | `for i in 1...5 {}` |

`Sources/01-Basics/Variables.swift`

```swift
let maximumNumberOfLoginAttempts = 10

if let actualNumber = convertedNumber {
    print("转换成功：\(actualNumber)")

    print("转换失败")
}

let colorToUse = userDefinedColorName ?? defaultColor
```

## 调试与安全

- 使用 `assert(condition, message)` 在调试环境中检查前置条件
- 使用 `precondition` 在发布版本也要确保条件成立
- 善用 `print` / `dump` 辅助调试

## 实战练习

1. **用户资料卡**：定义常量 `name`、变量 `age`、可选 `city`，输出友好描述
2. **温度转换器**：输入摄氏温度，计算华氏与开尔文
3. **成绩系统**：使用可选绑定解析 `String` → `Int`，并使用空合并提供默认值
4. **进阶**：封装一个函数，接收 `[String: String]`，使用 `guard` 验证必须字段
