## 学习目标

- 掌握类型转换 `is` / `as?` / `as!`
- 理解 `Any` 和 `AnyObject`
- 使用弱引用和无主引用解决循环引用
- 掌握可选链和嵌套类型

## 类型转换

```swift
if let movie = item as? Movie {
    print("电影：\(movie.name)")
}
```

## 内存管理

```swift
class Person {
    var apartment: Apartment?
}

class Apartment {
    weak var tenant: Person?
}
```
