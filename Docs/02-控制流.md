## switch 的威力

Swift 的 `switch` 非常强大：

- 支持多种类型（整数、字符串、枚举、元组）
- 支持区间匹配、值绑定、`where` 条件
- 每个 `case` 自动 break，要穿透需要 `fallthrough`

```swift
switch somePoint {
case (0, 0):
    print("原点")
case (_, 0):
    print("在 x 轴上")
case (0, _):
    print("在 y 轴上")
case let (x, y) where x == y:
    print("在对角线上")
default:
    print("其他位置")
}
```

## guard 的优势

`guard` 提前验证条件，保持主逻辑清晰：

```swift
func processPerson(_ person: [String: String]) {
    guard let name = person["name"] else {
        print("缺少姓名")
        return
    }
    print("处理：\(name)")
}
```
