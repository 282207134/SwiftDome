## 学习目标

- 理解闭包的概念与语法

- 掌握值捕获与逃逸闭包
- 熟悉 `map` / `filter` / `reduce`

## 闭包语法

```swift
{ (parameters) -> returnType in
    statements
}
```

简化：

```swift
reversedNames = names.sorted(by: { $0 > $1 })
```

## map / filter / reduce

```swift
let numbers = [1, 2, 3, 4, 5]
let doubled = numbers.map { $0 * 2 }
let evens = numbers.filter { $0 % 2 == 0 }
let sum = numbers.reduce(0, +)
```
