# 03 · 函数

参考：[Swift 中文文档 · 函数](https://doc.swiftgg.team/documentation/the-swift-programming-language/functions)

## 学习目标

- 定义和调用函数
- 使用参数标签、默认参数、可变参数
- 返回单个值、元组或无返回值
- 掌握 `inout` 参数与函数类型

## 核心语法

```swift
func greet(person: String, from hometown: String) -> String {
    return "你好，\(person)！很高兴你来自 \(hometown)。"
}

print(greet(person: "Bill", from: "Cupertino"))
```

## 高级特性

1. **参数标签**：外部标签 + 内部名称
2. **默认值**：`discount: Double = 0.1`
3. **可变参数**：`numbers: Double...`
4. **inout**：修改外部变量
5. **函数类型**：`(Int, Int) -> Int`

## 实战练习

1. **计算器函数**：加减乘除
2. **统计函数**：返回最大值、最小值、平均值（元组）
3. **高阶函数**：接收函数作为参数
