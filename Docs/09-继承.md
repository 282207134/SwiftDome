# 09 · 继承

参考：[Swift 中文文档 · 继承](https://doc.swiftgg.team/documentation/the-swift-programming-language/inheritance)

## 学习目标

- 创建基类和子类
- 使用 `override` 重写方法和属性
- 使用 `super` 调用父类实现
- 使用 `final` 防止重写

## 示例

```swift
class Vehicle {
    var currentSpeed = 0.0
    func makeNoise() {}
}

class Car: Vehicle {
    var gear = 1
    override func makeNoise() {
        print("Beep beep!")
    }
}
```

## 多态

```swift
let shapes: [Shape] = [Circle(radius: 5), Rectangle(width: 4, height: 6)]
for shape in shapes {
    print("面积：\(shape.area())")
}
```
