# 10 · 协议

参考：[Swift 中文文档 · 协议](https://doc.swiftgg.team/documentation/the-swift-programming-language/protocols)

## 学习目标

- 定义和遵循协议
- 使用协议作为类型
- 理解委托模式
- 掌握协议扩展

## 协议定义

```swift
protocol FullyNamed {
    var fullName: String { get }
}

struct Person: FullyNamed {
    var fullName: String
}
```

## 协议扩展

```swift
extension RandomNumberGenerator {
    func randomBool() -> Bool {
        return random() > 0.5
    }
}
```
