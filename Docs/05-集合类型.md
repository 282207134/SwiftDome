# 05 · 集合类型

参考：[Swift 中文文档 · 集合类型](https://doc.swiftgg.team/documentation/the-swift-programming-language/collectiontypes)

## 学习目标

- 使用数组（Array）存储有序列表
- 使用字典（Dictionary）存储键值对
- 使用集合（Set）存储无序唯一值
- 掌握集合的增删改查操作

## 快速对比

| 类型 | 特性 | 示例 |
| --- | --- | --- |
| Array | 有序、可重复 | `[1, 2, 3]` |
| Set | 无序、唯一 | `Set([1, 2, 2])` → `{1, 2}` |
| Dictionary | 键值对 | `["name": "Swift"]` |

## 集合操作

```swift
let oddDigits: Set = [1, 3, 5, 7, 9]
let evenDigits: Set = [0, 2, 4, 6, 8]

oddDigits.union(evenDigits)        // 并集
oddDigits.intersection(evenDigits)  // 交集
oddDigits.subtracting(evenDigits)   // 差集
```
