## 学习目标

- 理解结构体与类的异同
- 掌握值类型 vs 引用类型
- 使用恒等运算符 `===`
- 选择合适的类型

## 值类型 vs 引用类型

| 结构体 (Struct) | 类 (Class) |
| --- | --- |
| 值类型（复制） | 引用类型（共享） |
| 默认成员初始化器 | 需要自定义初始化器 |
| 不支持继承 | 支持继承 |
| 适用于数据模型 | 适用于复杂行为 |

## 实战案例

```swift
struct Point {
    var x: Double
    var y: Double
}

class BankAccount {
    var balance: Double
    init(balance: Double) {
        self.balance = balance
    }
}
```
