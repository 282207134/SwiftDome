# 贡献指南

感谢你对 Swift 学习项目的关注！为了保持一致性，请遵循以下约定：

## 开发流程

1. Fork 或创建新分支
2. 提交代码前，请自检编译或运行示例
3. 提交带有意义的 commit message

## 代码规范

- 所有 Swift 代码必须包含中文注释
- 遵循 Swift API Design Guidelines
- 文件名使用 `UpperCamelCase.swift`

## 文档规范

- 所有 Markdown 文件需使用二级及以下标题划分内容
- 提供示例代码块，并标注语言
- 如引用外部文档，请附上链接

## 目录结构

- `Sources/`：示例代码（按章节分类）
- `Docs/`：学习文档
- `Examples/`：综合项目

## 提交说明

- 新增文件请更新相关 README 或文档
- 如涉及重大改动，请在 PR 中详细说明

欢迎提交 Issue 或 Pull Request，与大家一起完善学习资源！
